Підтримувані версії
Ми регулярно випускаємо оновлення безпеки лише для актуальних гілок. Старі версії можуть отримувати виправлення лише у виняткових випадках.

Версія	Підтримка
5.1.x	✅ активна підтримка
5.0.x	❌ не підтримується
4.0.x	✅ активна підтримка
< 4.0	❌ не підтримується


Повідомлення про вразливості
Ми серйозно ставимося до безпеки й вдячні кожному, хто допомагає зробити проєкт надійнішим.

Куди писати: надсилай деталі на нашу офіційну пошту безпеки або через GitHub Security Advisories.

Що включити: чіткий опис проблеми, кроки для відтворення, можливий вплив та пропозиції щодо виправлення.

Час відповіді: ми намагаємось підтвердити отримання повідомлення протягом 48 годин.

Процес:

Якщо вразливість підтверджена, ми створюємо приватний issue й працюємо над виправленням.

Ти отримаєш оновлення про статус (прийнято / відхилено / виправлено).

Після випуску патчу ми публікуємо повідомлення про проблему та рішення.

Принципи
Ми не розкриваємо деталі вразливості публічно, доки не буде доступне виправлення.

Автори повідомлення отримують подяку у changelog або advisory.
